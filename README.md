# UX Portfolio

A modern, responsive portfolio website for showcasing UX/UI design work, built for deployment on GitHub Pages.

## 🎨 About This Project

This portfolio website is designed to showcase UX/UI design projects, case studies, and professional experience. It's built to be:

- **Responsive** - Works beautifully on desktop, tablet, and mobile
- **Fast** - Optimized for quick loading times
- **Accessible** - Following web accessibility best practices
- **Easy to maintain** - Simple structure for easy updates

## 🚀 Quick Start

### Prerequisites
- A GitHub account
- Basic knowledge of HTML/CSS (optional but helpful)

### Getting Started

1. **Clone this repository**
   ```bash
   git clone https://github.com/butteredbagel/UX-Portfolio.git
   cd UX-Portfolio
   ```

2. **Customize your portfolio**
   - Update personal information in `index.html`
   - Add your projects to the portfolio section
   - Replace placeholder images with your work
   - Update the color scheme in `styles.css` to match your brand

3. **Test locally** (optional)
   - Open `index.html` in your web browser to preview changes

## 🚢 Deployment to GitHub Pages

1. **Enable GitHub Pages**
   - Go to repository Settings
   - Scroll to "Pages" section
   - Select "Deploy from a branch"
   - Choose "main" branch and "/ (root)" folder
   - Click "Save"

2. **Your site will be available at:**
   ```
   https://butteredbagel.github.io/UX-Portfolio/
   ```

3. **Custom domain (optional)**
   - Add a `CNAME` file with your custom domain
   - Configure DNS settings with your domain provider

## 🎨 Customization Guide

### Colors and Branding
- Update CSS variables in `css/style.css` for consistent theming
- Replace favicon in root directory
- Update meta tags for social media sharing

### Adding Projects
1. Add project images to `images/portfolio/`
2. Update portfolio grid in `portfolio.html`
3. Create individual project pages in `projects/` folder
4. Include case study format: Problem → Research → Solution → Results

### Content Guidelines
- **Hero section**: Clear value proposition and what you do
- **About**: Your design philosophy and background
- **Portfolio**: 4-6 best projects with strong visuals
- **Case studies**: Problem, process, solution, and impact
- **Contact**: Professional email and LinkedIn
